# Assets

Optional PNG files:

- `taffy-character.png`: character art for the left hugging layer.
- `taffy-character-card.png`: rounded sticker version of the character art; loaded first when present.
- `taffy-logo.png`: logo or title art for the top of the data card.
- `taffy-headshot.png`: plugin logo image.
- `taffy-headshot.ico`: window, tray, and desktop shortcut icon.

Transparent PNGs work best for the character layer.

Before publishing this repository publicly, verify that you have permission to redistribute any
character artwork, Bilibili avatar images, logos, or fan art included in this folder. If not,
replace these assets with your own images and keep only the expected filenames.
