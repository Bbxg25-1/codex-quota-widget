# Codex 额度查看小窗口公开发布加固设计

## 目标

把现有个人自用脚本整理为适合公开发布的 Windows 小工具，同时保持现有额度读取、单实例、自动刷新和随 Codex 显示/隐藏等功能不变。

## 发布策略

- Git 仓库不再分发永雏塔菲角色图、Bilibili 头像、Logo 或其他授权不明确的第三方素材。
- 当前电脑上的塔菲素材先复制到 `%LOCALAPPDATA%\CodexQuotaWidget\assets`，继续作为本机私有自定义素材使用。
- 程序优先读取本机私有素材目录；缺少私有素材时使用仓库内原创通用图标。
- README 截图只展示原创通用视觉，不出现第三方角色素材。

## 安全与隐私

- README 顶部醒目说明读取 `%USERPROFILE%\.codex\auth.json` 的目的和范围。
- access token 仅作为请求 `chatgpt.com` 官方个人资料接口的 Authorization 请求头。
- token 不写入日志、不写入文件、不显示在 UI、不发送到第三方服务器。
- 本机 Codex app-server 只通过 `127.0.0.1` 临时连接，读取额度窗口后立即关闭。

## 用户体验

- README 提供中英双语的预览、三步快速开始、PowerShell 执行策略说明和 FAQ。
- 将技术状态映射为能直接指导用户处理问题的中文提示。
- 保留现有窗口设计和主要布局，不进行大规模 UI 重构。

## 开源与发布

- 代码使用 MIT License。
- 第三方自定义素材不受 MIT License 覆盖，用户自行负责素材授权。
- 准备 `v0.2.0` ZIP 和中英双语更新说明，但本次不创建 GitHub Release。

## 验收

- 仓库不跟踪塔菲相关素材。
- 本机塔菲版仍可通过私有素材目录运行。
- README 包含安全说明、截图、快速开始、FAQ、Bypass 解释和 License。
- 常见失败状态显示明确解决办法。
- 插件校验、脚本测试、快照读取、单实例和发布包检查通过。
